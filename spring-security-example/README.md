# Dependencies
- Spring
- Spring security
- Spring Data (JPA / Hibernate)
- Spring dev-tools
- Thymeleaf 3
- Postgresql
- log4j / logback

# Some useful commands

1. Pull static library
```
$ cd src/main/resources/static
$ npm install
```

2. Generate gradle support for idea
```
$ ./gradlew idea
```

3. Build
```
$ ./gradlew build
```

4. Run
```
$ ./gradlew bootRun
```
